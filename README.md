
# azure-activedirectory-library-for-python
Python library and sample app for accessing Azure AD

This is an early, pre-release version of the library.

## Usage

1. Update the config.py, with appropriate values. It has client id, client secret, redirect uri, storage location etc.
2. Keep the sample project as default project of the solution.
3. Run the application, once python window shows it, running. Open the url : http://localhost:5000 in your browser and you are ready to use the sample.

## Samples and Documentation
The `samples` folder contains a sample implementation of the library demonstrating basic authentication in several different ways.
